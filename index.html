Informações sobre Stardew Valley

Sobre

Esse app vai checar no site Stardew Valley com seu save file por conquistas e segredos que você ainda tem por fazer ou fez. No estado atual está checando por 38 conquistas.

Está hosteado em minha github page https://thousanddaggerss.github.io/

Local Para o Save File

Selecione o arquivo salvo para checar:

Use o nome do arquivo completo com o nome do seu fazendeiro e os 9 números que estão nele (exp.: Fazendeiro_564813548); não use o arquivo SaveGameInfo que não contém todas as informações que precisamos.

Local padrão onde achar o arquivo:

  • Windows: %AppData%\StardewValley\Saves\
